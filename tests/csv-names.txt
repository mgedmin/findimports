Producing import graphs listing all imported names
==================================================

findimports can produce a detailed list of all imported names

    >>> from findimports import main
    >>> exitcode = main(['findimports', '--tabular', sample_tree])
    apple,os
    apple,os.path
    apple,sys
    apple,sys
    box.cat,gc
    box.cat,box.yarn
    box.cat,decoy
    orange,gc
